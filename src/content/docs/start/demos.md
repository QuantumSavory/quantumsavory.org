# Interactive Demos

These demos are based on the [QuantumSavory.jl examples folder](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples). Some are already hosted. Others are placeholders for future deployments.

## Hosted Now

- [State Explorer](https://areweentangledyet.com/state_explorer/): inspect parameterized two-qubit resource states before using them in simulations. Source: [examples/state_explorer](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/state_explorer).
- [Color-Center Cluster States](https://areweentangledyet.com/colorcentermodularcluster/): simulate GHZ and cluster-state generation in color-center memories. Source: [examples/colorcentermodularcluster](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/colorcentermodularcluster).
- [Repeater Chain Congestion](https://areweentangledyet.com/congestionchain/): tune repeater-chain length, memory size, fidelity, coherence, generation time, and swap duration. Source: [examples/congestionchain](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/congestionchain).
- [Simple Entanglement Switch](https://areweentangledyet.com/simpleswitch/): star-network switch simulation with client request rates, backlog, and completed-pair plots. Source: [examples/simpleswitch](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/simpleswitch).
- [QEPOptimize Pluto Demo](https://doc.quantumsavory.org/QEPOptimize.jl/example/pluto.html): interactive generation of optimized entanglement-purification circuits.
- [Quantikz Live Renderer](https://quantikz.krastanov.org/): render quantum circuits in the browser.

## Placeholder Slots

- First-generation repeater with Genqo sources: high-level repeater-chain simulation with source-parameter controls. Source: [examples/firstgenrepeater_v2](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/firstgenrepeater_v2).
- Repeater grid with asynchronous messages: grid-network entanglement distribution with local knowledge and message passing. Source: [examples/repeatergrid](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/repeatergrid).
- StatesZoo REST API: density matrices and metadata for resource-state models. Source: [examples/states_rest_server](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/states_rest_server).
- QTCP tutorial: chain and grid examples for quantum transport control protocols. Source: [examples/qtcp_tutorial](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/qtcp_tutorial).
- GHZ projection setup: hub-based projection from Bell pairs to multi-party GHZ resources. Source: [examples/GHZsetup](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/GHZsetup).
- Assisted continuous-variable teleportation: Gabs.jl-based teleportation with homodyne measurements and classical messages. Source: [examples/assisted_cvteleportation](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/assisted_cvteleportation).
- Graph-state constructor: scheduled parallel entanglement rounds and graph-state fusion. Source: [examples/graphstate](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/graphstate).
- MBQC purification: measurement-based purification with message tags and entanglement tracking. Source: [examples/purificationMBQC](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/purificationMBQC).
- Piecemaker GHZ switch: switch-centered GHZ distribution for client links. Source: [examples/piecemakerswitch](https://github.com/QuantumSavory/QuantumSavory.jl/tree/master/examples/piecemakerswitch).
